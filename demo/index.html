The YaoApi Client library retrieves data of all levels from the RESTFUL API endpoints.
Restful API Server: http://192.168.0.124:3002/api/v1

avaiable endpoints:

http://192.168.0.124:3002/api/v1/assets/
http://192.168.0.124:3002/api/v1/assets/1
http://192.168.0.124:3002/api/v1/categories
http://192.168.0.124:3002/api/v1/categories/1/
http://192.168.0.124:3002/api/v1/categories/1/subcategories
http://192.168.0.124:3002/api/v1/categories/4/parent
http://192.168.0.124:3002/api/v1/categories/1/items

so we can operate all level of data (Asset, Category, SubCategory, PDF)
just with even one object.

a way to call Yao API

        var yao1 = new Yao.YaoApi();
        yao1.assetData(1).then(function (assetData) {
          console.log(assetData);
        }).catch(function (error) {
          console.log(error);
        })

example:

        // display category list 
        cat_list = assetData.categories;
        cat_list.forEach(function(cat){
          console.log(cat.name);
        });
